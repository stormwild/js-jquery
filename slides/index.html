JavaScript History

  • Developed by Brendan Eich for Netscape in 1995
  • Standardized in 1997 as ECMA-262 and name ECMAScript
  • Modern Browsers largely conform to ECMAScript 3 & 5
  • Requires a host environment
    • Browser (client-side)
    • NodeJs (server-side)
  • Dynamic, object-oriented, scripting language
